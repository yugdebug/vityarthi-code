# Project Statement
## Problem statement
In the digital age most of the existing QR generators are web-based and requiring an internet connection. A standard QR code is typically monochromatic and lack a visual appeal to them. Users want a simple, offline and secure desktop solution to create customized QR codes with brand colors and logos without requiring some technical knowledge.

## Scope of the project

1. Offline input : The system will accept text or URL strings from the user.

2. Image Processing: The application will handle visual customization. It includes features to overlay logos and modify pixel colors.

3. File Output: The scope includes rendering the final output as a standard image file (PNG) and saving it to the user's local storage.

4. being intrigued by easter eggs 

## Limitations
The project is limited to generating static QR codes and does not include features for analytics or mobile app interface.

## Target users

1. Small Business Owners: To create QR codes for marketing materials, menus or business cards.

2. Event Organizers: For generating quick-scan links for tickets or event schedule.

3. General Users: Anyone needing a qr.

4. people who love history
